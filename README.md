# 基金智管家

支付宝场外基金智能助手，专注基金投资，简单易懂。

## 功能特点

- 📊 **实时估值**：盘中实时获取基金估值
- 💼 **持仓管理**：记录买入成本、计算盈亏、止盈提醒
- 📈 **净值走势**：查看历史净值和均线分析
- 💡 **操作建议**：智能生成买卖建议
- 🔍 **基金搜索**：快速查询基金信息
- ⏰ **14:30 提醒**：重要时间节点提醒
- 🛡️ **导入引导**：首次使用自动提示导入备份数据

## 重要提示

⚠️ **数据存储说明**：Streamlit Cloud 免费版每次重启后数据会丢失！

请务必：
1. 每次使用后点击 **「一键导出持仓」** 下载备份
2. 重启后应用会自动提示导入，上传备份文件恢复数据
